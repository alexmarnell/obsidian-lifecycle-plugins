# obsidian-lifecycle-plugins

Claude Code plugins for the [Obsidian Lifecycle System](https://github.com/alexmarnell/obsidian-lifecycle).

## Installation

Add the marketplace:

```
/plugin marketplace add alexmarnell/obsidian-lifecycle-plugins
```

Then install individual plugins:

```
/plugin install bridge@obsidian-lifecycle-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [bridge](plugins/bridge/) | Capture knowledge and recall vault notes across workspaces |
